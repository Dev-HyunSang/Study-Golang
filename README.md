## [fmt Format](https://golang.org/pkg/fmt/)
- ```%t```  bool 
- ```%b``` 2진수
- ```%c``` 문자
- ```%d``` 10진수
- ```%o``` 8진수
-``` %x``` 16진수, a~f
- ```%X``` 16진수, A~F
- ```%f```  고정소수점
- ```%F``` 고정소수점
- ```%e``` 지수 표현, e
- ```%E``` 지수 표현, E
- ```%g``` 간단한 실수 표현
- ```%G``` 간단한 실수 표현
- ```%s``` 문자열
- ```%p``` 포인터
- ```%U``` 유니코드
- ```%T``` 타입
- ```%v``` 모든 형식
- ```%#v``` 형식을 구분할 수 있게
